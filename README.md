Analiza Econometrică a Indicatorilor Macroeconomici
Proiect realizat în EViews pentru analiza și prognoza seriilor de timp (IAPC, PIB, Consum) în România.
Analize: Sezonalitate, modele ARIMA, netezire Holt-Winters, cauzalitate Granger și modele VAR.
Tehnici: Teste Unit Root (ADF), filtrare Hodrick-Prescott, analiză de cointegrare și funcții impuls-răspuns.
Baza de date: Eurostat și TEMPO Online (INS).
